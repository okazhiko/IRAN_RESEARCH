# インサイト
- ニュース事項による日本の電力市場への影響（短期：数日〜1週間）
	- OILは `95.08` と前日比 `-7.03%`、先週比 `-11.04%` まで下落し、ホルムズ再開期待による原油リスクプレミアムの剥落と整合的です。燃料面だけを見ると、数日〜1週間のJEPX上振れ圧力は前週より弱まっています。
	- JEPXシステムプライスは `12.48` 円/kWhで前日比 `37.14%` 上昇しましたが、先週比・先月比はいずれも `-18.91%` です。燃料価格の急落観測が出ても、日々の需給要因でスポットは反発し得ますが、危機ピーク型の急騰には戻っていません。
	- LNGは最新値が `2026-05-01` でデータラグがありますが、`2761` と前日比 `-3.93%`、先週比 `2.45%` です。短期では、原油急落が電力市場心理を冷やす一方、ホルムズでの船舶攻撃や米イラン合意未成立が残るため、下落一方向ではなく不安定な低位反発が想定されます。
- ニュース事項による日本の電力市場への影響（長期：1ヶ月〜半年）
	- ホルムズ再開が実現すれば、OILの先月比 `-9.51%` という下落は発電燃料費の見通し改善につながります。ただし、APが示すように米政権の方針はなお不安定で、交渉不成立時の再爆撃リスクが長期契約や調達ヘッジの必要性を残します。
	- 非ホルムズ系LNGでは、米国案件の長期SPAやLNG船の長期契約化が進んでいます。これは日本の調達多角化に資する一方、輸送船・ガスタービンなどの制約が価格を硬直化させるため、1ヶ月〜半年で安価な代替供給が一気に増えるとは見込みにくいです。
	- 日本のUAE共同備蓄拡大は、ホルムズ不安に対する供給安全保障策として有効です。JEPXは過去最高値比でシステム `-80.52%` と低い水準にありますが、燃料調達の長期リスクが消えたわけではなく、価格低下局面でも備蓄・代替調達・燃料費ヘッジを維持する判断が妥当です。
